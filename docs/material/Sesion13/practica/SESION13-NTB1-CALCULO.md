# Sesion 13: Ejercicios de Calculo Multivariable

Material intermedio-avanzado orientado a IA. Cada ejercicio se desarrolla paso a paso con interpretación aplicada.

## Ejercicios resueltos paso a paso

### Ejercicio 1: Gradiente multivariable caso 1

**Enunciado (contexto IA):** Para $f(u,v)=u^2+3v^2$, calcula $
abla f$ en $(1,2)$.

**Solución paso a paso:**

1) Parciales: $\partial f/\partial u=2u$, $\partial f/\partial v=6v$.
2) Sustituye valores.
3) Resultado: $
abla f=(2,12)$.
4) IA: vector base para GD y Adam.
