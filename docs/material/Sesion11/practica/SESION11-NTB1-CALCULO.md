# Sesion 11: Ejercicios de Limites y Continuidad

Material intermedio-avanzado orientado a IA. Cada ejercicio se desarrolla paso a paso con interpretación aplicada.

## Ejercicios resueltos paso a paso

### Ejercicio 1: Limite racional caso 1

**Enunciado (contexto IA):** Calcula $\lim_{x\to 2} \frac{x^2-4}{x-2}$ y explica continuidad.

**Solución paso a paso:**

1) Factoriza: $x^2-4=(x-2)(x+2)$.
2) Cancela factor comun para $x
eq 2$.
3) Evalua limite: 4.
4) IA: simplificar expresiones evita inestabilidad numerica.
